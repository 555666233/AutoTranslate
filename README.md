# 自动翻译器（文本翻译）

> 一款专注于 **小而精** 的文本文件批量自动化翻译工具，专为Sakura模型适配优化，支持多种格式的文本文件。

## 🎯 项目简介

支持TXT文本文件、SRT字幕文件、MTool导出的JSON游戏文本文件、XUnity.AutoTranslator缓存的TXT游戏文本文件、Translator++导出的Excel游戏文本文件的批量自动化 **免费** 翻译器！
-  TXT文本：文件类型为 .txt 的纯文本文件，比如小说。
-  SRT字幕：文件类型为 .srt 的标准字幕文件。
-  MTool导出的游戏文本：MTool是一款支持多种RPG引擎游戏的翻译工具，通过它可以导出游戏文本，翻译后也可通过它将译文导回游戏中。[了解 MTool](https://mtool.app/)
-  XUnity.AutoTranslator缓存的游戏文本：XUnity.AutoTranslator是一款针对Unity引擎游戏的自动翻译插件。[了解 XUnity.AutoTranslator](https://github.com/bbepis/XUnity.AutoTranslator)
-  Translator++导出的Excel游戏文本： Translator++是一款支持多种RPG引擎游戏的翻译工具，通过它不仅能导出游戏文本，而且能将译文内嵌回游戏。[了解 Translator++](https://dreamsavior.net/)

## ✨ 特性亮点

- **📊 界面简洁，操作简单**：提供清晰直观的操作界面。
- **🤖 AI 翻译**：支持所有兼容 OpenAI 标准格式接口的大语言模型API，包括线上API和本地部署的API。
- **🎯 文本清洗和富文本过滤**：减少干扰文本，提升翻译的质量和效率。
- **📖 支持术语表功能**：支持术语表定制，能确保存在一词多译法的词语在整个文本中的翻译保持一致。
- **📁 批量翻译**：支持同类型的多文件排队翻译，方便省心。
